🔌 API Endpoints

Platform, 50+ endpoint ile kapsamlı bir REST API sunar:

Kimlik Doğrulama

POST /api/auth/login - Kullanıcı girişi
POST /api/auth/2fa/login - 2FA ile giriş
GET /api/auth/me - Kullanıcı bilgileri

Sipariş Yönetimi

GET /api/orders - Sipariş listesi
GET /api/orders/stats/summary - Sipariş istatistikleri
POST /api/orders - Yeni sipariş oluşturma

WooCommerce Entegrasyonu

GET /api/woo/stores - Mağaza listesi
POST /api/woo/webhooks/:id - Webhook alıcı

🛡️ Güvenlik Özellikleri

Şifreleme

  • AES-256-GCM envelope encryption
  • JWT RS256 (production) / HS256 (development)
  • bcrypt password hashing (cost factor 10)
  • SHA256 session token hashing

Kimlik Doğrulama

  • Multi-factor authentication (TOTP)
  • Session management with fingerprinting
  • Account lockout after failed attempts
  • Role-based access control (RBAC)

Ağ Güvenliği

  • Rate limiting at multiple layers
  • Security headers (HSTS, CSP, X-Frame-Options)
  • SQL injection prevention via Prisma
  • XSS protection

🚀 Deployment ve Kurulum

Platform, Docker Compose ile kolay kurulum sağlar:

Sunucuya Bağlanın

ssh root@your-server-ip

Repository'yi Klonlayın

git clone https://github.com/fulexo/panel.git /opt/fulexo cd /opt/fulexo

Hızlı Kurulum Script'ini Çalıştırın

chmod +x scripts/quick-install.sh ./scripts/quick-install.sh

Kurulum Tamamlandı!

Panel: https://your-domain.com
API: https://api.your-domain.com
API Docs: https://api.your-domain.com/docs
